# crisby-battery-
AA veya AAA pilleri ölçmek için proje

# Malzemeler 
 1 adet NodeMCU\
 1 adet pil yuvası\
 1 adet Nokia 5110 LCD\
 11 adet jumper kablo\
 1 adet breadboard

# Kullanılan Kütüphaneler
<Arduino.h>\
<Wire.h>\
<SPI.h>\
<Adafruit_PCD8544.h> --Nokia 5110 LCD kullanmak için--\
<stdlib.h>\
"battery.h"
